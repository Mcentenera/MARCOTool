# MARCOTool
A pipeline for telescope, photonic lantern and instrument design

<img src="Figures/logo.png" width="250" height="auto">

## Installation instruction

Requirements:

-  Python (2.7 or 3.6+), numpy, scipy, matplotlib, pyqt5, seaborn

## How to start

Create an environment as:

```bash
conda create -n marcotool python=3.10
```

Activate the environment as:

```bash
conda activate marcotool
```

Go to the folder where you download the files and install dependencies as:

```marcotool
pip install -r requirements.txt
```

Execute as:

```marcotool
python MARCOTool.py
```
